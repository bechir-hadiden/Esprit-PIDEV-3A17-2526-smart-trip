
SmartTrip – Application de Gestion d'Agence de Voyage
Présentation
Ce projet a été développé dans le cadre du PDEV – 3ème année Ingénierie à Esprit School of Engineering (Année académique 2025–2026). Il s'agit d'une application web moderne de gestion d'agence de voyage construite avec Symfony 6, permettant aux utilisateurs de parcourir des destinations, réserver des voyages, gérer les bus et les passagers, et suivre les réservations en temps réel.

Fonctionnalités principales

Interface moderne et responsive avec Twig, CSS3 et JavaScript
Gestion des voyages : ajout, modification, suppression de voyages
Gestion des bus : suivi des bus disponibles, capacité et itinéraires
Gestion des passagers : ajout, modification et suivi des points de dépôt/prise en charge
Réservations : trajets simples et allers-retours avec confirmation en temps réel
Statistiques dynamiques et tableaux de bord (nombre de voyages, passagers, revenus)
Gestion des utilisateurs : rôles, authentification et sécurité via Symfony Security
Historique des réservations pour chaque client
Notifications et alertes pour les réservations ou mises à jour
Recherche avancée : par destination, date ou type de voyage
Export des données en PDF ou CSV pour les rapports
Gestion des tarifs et promotions


Stack technique

PHP 8.2
Symfony 6.4
MySQL pour la base de données
Doctrine ORM pour la gestion des entités
Twig pour le moteur de templates
Composer pour la gestion des dépendances
API Platform (optionnel) pour les endpoints REST
Webpack Encore pour les assets front-end
